# フォーム全体の見た目調整

## 結論
フォームは「幅・余白・並び」をそろえると一気に見やすくなります。

## 何が起きているか
入力欄の幅やラベルの位置がバラバラだと、読みにくくなります。そろえるだけで整った印象になります。

## 悪い例（デモ）
幅がバラバラで、視線が迷います。

デモ: `/demos/form-layout.html`

```css
input { width: auto; }
```

## 良い例（デモ）
幅と余白を統一して読みやすくします。

デモ: `/demos/form-layout.html`

```css
input { width: 100%; }
.form { max-width: 320px; display: grid; gap: 10px; }
```

## なぜそうなるか
人は「そろった形」を読みやすいと感じます。幅と余白の統一が効果的です。

## よくあるミス
- 入力欄の幅を固定して小さくなる
- ラベルと入力欄の距離がバラバラ
