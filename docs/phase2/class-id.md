# 補足：classとidの違い

## 結論
`class`は「何度でも使える名札」、`id`は「1回だけの名札」です。

## 何が起きているか
`class`は同じ見た目のグループに使います。`id`はページ内で1つだけの特別な場所に使います。

## 悪い例（デモ）
同じ見た目にしたいのに、`id`を複数使うと管理が大変です。

デモ: `/demos/class-id.html`

```html
<p id="label">お知らせ</p>
<p id="label">告知</p>
```

## 良い例（デモ）
繰り返すなら`class`を使います。

デモ: `/demos/class-id.html`

```html
<p class="label">お知らせ</p>
<p class="label">告知</p>
```

## なぜそうなるか
`id`は「このページで1つだけ」と決めるためのものです。だから複数に使うとルールが壊れます。

## よくあるミス
- `id`を何個も使ってしまう
- `class`と`id`を混ぜて覚えてしまう
