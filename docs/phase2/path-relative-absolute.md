# 相対パスと絶対パス（./ と ../）

## 結論
相対パスは「今いる場所からの道順」。`./`は今のフォルダ、`../`は1つ上のフォルダです。

## 何が起きているか
ブラウザはファイルの場所をたどります。道順が違うと、CSSや画像が見つからず読み込めません。

## 悪い例（デモ）
1つ上のフォルダにあるCSSなのに、`./`を使ってしまうと見つかりません。

デモ: `/demos/path-relative-absolute.html`

```html
<link rel="stylesheet" href="./styles/main.css" />
```

## 良い例（デモ）
`../`で1つ上に戻ると、正しい場所にたどり着きます。

デモ: `/demos/path-relative-absolute.html`

```html
<link rel="stylesheet" href="../styles/main.css" />
```

## なぜそうなるか
パスは地図です。1文字違うだけで別の場所になり、ファイルが見つからなくなります。

## よくあるミス
- `./`と`../`を逆にする
- フォルダ名の大文字小文字を間違える
