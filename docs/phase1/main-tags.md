# HTMLの主要タグまとめ

## 結論
よく使うのは「見出し・段落・リスト・画像・リンク・ボタン」です。まずここだけ覚えればOKです。

## 何が起きているか
HTMLは部品の集合です。部品の意味が分かれば、ページの構造が自然に作れます。

## 悪い例（デモ）
全部を`div`で作ると、意味が伝わりにくくなります。

デモ: `/demos/main-tags.html`

```html
<div>新着記事</div>
<div>HTMLの基礎</div>
<div>CSSの基礎</div>
<div>もっと読む</div>
```

## 良い例（デモ）
意味のあるタグを使うと、読み手にもブラウザにも親切です。

デモ: `/demos/main-tags.html`

```html
<h2>新着記事</h2>
<ul>
  <li>HTMLの基礎</li>
  <li>CSSの基礎</li>
</ul>
<a href="#">もっと読む</a>
```

## なぜそうなるか
タグは「これは見出し」「これはリンク」と伝えるラベルです。見た目だけでなく、意味がそろいます。

## よくあるミス
- 全部`div`にしてしまい、CSSが複雑になる
- 見出しを`p`で作ってしまう
