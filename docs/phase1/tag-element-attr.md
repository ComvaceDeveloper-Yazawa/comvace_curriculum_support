# タグ・要素・属性とは

## 結論
タグは部品名、要素はタグで囲んだ完成品、属性は追加の情報です。

## 何が起きているか
`<p>`のようなタグで囲むと要素になります。属性は「この要素はどんな性格か」を付け足します（例: class）。

## 悪い例（デモ）
属性が無いので、CSSで狙い撃ちできず見た目がそろいません。

デモ: `/demos/tag-element-attr.html`

```html
<p>今日の目標</p>
<p>HTMLとCSSの役割</p>
<p>なぜ縦に並ぶのか</p>
```

## 良い例（デモ）
`class`属性でまとめて指定でき、同じ見た目にできます。

デモ: `/demos/tag-element-attr.html`

```html
<p class="goal">今日の目標</p>
<p class="goal">HTMLとCSSの役割</p>
<p class="goal">なぜ縦に並ぶのか</p>
```

## なぜそうなるか
CSSは「どの要素に当てるか」を決めるための目印が必要です。属性がその目印になります。

## よくあるミス
- タグと要素を同じ意味だと思って混乱する
- 属性を書き忘れてCSSが効かない
