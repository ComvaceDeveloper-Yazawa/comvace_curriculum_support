# TypeScript 基礎

## 先に知っておきたい：TypeScriptを学ぶメリット

- プログラミングの基礎が身に付く（変数・関数・条件分岐が整理される）
- 早い段階で間違いに気づける（書いた瞬間にエラーが出る）
- フロントだけでなく、フルスタックにも近づく（Node.js + TypeScript）
- チーム開発で安心して書ける（ミスが減り、意図が伝わる）

この章では、
「型を付けてミスを減らす」考え方を学びます。

## この章で身につくこと
- TypeScript の役割が分かる
- JavaScript との違いが分かる
- 実務で安心して書ける土台ができる

## 学び方の目安
最初は「ここに文字が入る」「ここに数が入る」
この感覚がつかめれば十分です。
具体的なチュートリアルは順次追加します。
