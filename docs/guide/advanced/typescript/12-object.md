# 第12章：オブジェクトってなに？

## この章で身につくこと

- オブジェクトが「名前付きのデータ」だと分かる
- キーと値の関係が分かる

## 12-1 オブジェクトは「名前付きの箱」

```ts
const user: { name: string; age: number } = {
  name: "Taro",
  age: 22,
};
```

`name` と `age` のように、
「名前（キー）と値」をセットで持てます。

## 12-2 取り出し方

```ts
const userName = user.name;
```

`user.name` で「name の値」を取り出せます。

## 12-3 変更のしかた

```ts
user.age = 23;
```

`age` の値を 23 に更新しています。

## 12-4 配列との違い

- 配列: 順番が重要なリスト
- オブジェクト: 名前でアクセスする箱

「人」や「商品」のように項目が決まっているときに使います。

## ❌ よくあるミス

- `user.name` のつもりで `user.naem` と打ち間違える
- 存在しないキーにアクセスして `undefined` になる
- 配列とオブジェクトを混同してしまう
