# 第11章：配列ってなに？

## この章で身につくこと

- 配列が「同じ種類のデータの並び」だと分かる
- 基本の使い方が分かる

## 11-1 配列は「リスト」

同じ種類のデータをまとめて持つ箱です。

```ts
const scores: number[] = [80, 90, 75];
```

`scores` は「数字のリスト」です。

## 11-2 取り出し方

```ts
const first = scores[0];
```

`[0]` は「1番目」を意味します。
配列は 0 から数えます。

## 11-3 追加と長さ

```ts
scores.push(100);
const count = scores.length;
```

- `push`: 最後に追加する
- `length`: 要素の数

`count` には配列の数が入ります。

## 11-4 配列はよく使う

- 商品の一覧
- ユーザーの一覧
- メッセージの履歴

「同じ種類のものがたくさんある」ときに配列を使います。
