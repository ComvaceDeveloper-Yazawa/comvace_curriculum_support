# 第1章：コンピュータとプログラムの基本

## この章で身につくこと

- プログラムが「命令の集まり」だと分かる
- 変数・条件分岐・繰り返しの役割が分かる
- 「入力 → 処理 → 出力」の考え方が分かる

## 1-1 プログラムは「お願いの文章」

コンピュータは、こちらが書いた命令を順番に実行します。
だからプログラムは「お願いの文章」です。

- まず何をするか
- その次に何をするか
- 条件が違えばどうするか

この順番さえ守れば、どんな人でも作れるようになります。

## 1-2 基本は「入力 → 処理 → 出力」

```ts
const price = 1200; // 入力
const tax = price * 1.1; // 処理
console.log(tax); // 出力
```

- `price` は入力（もとの値）
- `tax` は処理結果（計算された値）
- `console.log` は出力（画面に出す）

まずはこの流れだけ覚えれば OK です。

## 1-3 変数・条件分岐・繰り返し

### 変数

値に名前をつけて覚えておく箱です。

```ts
const userName = "Sora";
```

`userName` という箱に文字を入れています。

### 条件分岐

「もし○○なら、こうする」です。

```ts
const age = 20;

if (age >= 18) {
  console.log("大人です");
}
```

`age` が 18 以上なら「大人です」と出します。

### 繰り返し

同じ処理を何回もするための書き方です。

```ts
for (let i = 0; i < 3; i++) {
  console.log("こんにちは");
}
```

「こんにちは」を 3 回表示します。

## 1-4 今日はこの感覚でOK

- 命令を順番に書く
- 入力→処理→出力を意識する
