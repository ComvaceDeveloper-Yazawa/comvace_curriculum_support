# 第2章：コマンドってなに？（ターミナル入門）

## この章で身につくこと

- コマンドが「PCへの命令」だと分かる
- ターミナルの役割が分かる
- よく使う最低限のコマンドが分かる

## 2-1 コマンドは「PCにお願いする短い文」

普段はクリックで操作しますが、
文字で命令する方法がコマンドです。

例えば「今いる場所を教えて」などを文字で伝えます。

## 2-2 ターミナルってなに？

ターミナルは「コマンドを打つ場所」です。
Windows なら PowerShell、Mac なら ターミナル を使います。

## 2-3 よく使うコマンド

```bash
pwd
ls
cd folder-name
```

- `pwd`: 今いる場所を表示する
- `ls`: 今いる場所のファイル一覧を出す
- `cd`: 目的のフォルダに移動する

例えば `cd my-project` は、`my-project` フォルダへ移動します。

## 2-4 コマンドは怖くない

やっていることは「場所を確認して移動する」だけです。
分からなくなったら `pwd` で現在地を見れば OK です。

## ❌ よくあるミス

- `cd` で存在しないフォルダ名を打ってしまう（大文字・小文字も区別される）
- `cd my project` のようにスペースのある名前で失敗する
- どこにいるか分からなくなり、違う場所で作業してしまう
