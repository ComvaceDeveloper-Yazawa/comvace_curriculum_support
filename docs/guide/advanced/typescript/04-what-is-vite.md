# 第4章：Viteってなに？

## この章で身につくこと

- Vite が「開発を早くする道具」だと分かる
- Vite を使う理由が分かる

## 4-1 Viteは開発用の道具

Vite（ヴィート）は、
HTML/CSS/TypeScript を動かしやすくする道具です。

- すぐに開発サーバーが立つ
- 変更が即反映される
- TypeScript がそのまま使える

「面倒な準備を全部やってくれる仕組み」だと思えば OK です。

## 4-2 なぜViteを使う？

昔は自分で設定が必要でしたが、
Vite を使うとすぐに開発が始められます。

初心者でも「書くこと」に集中できます。

## 4-3 今日は概要だけ覚える

- Vite = 開発を楽にする道具
- すぐにブラウザで確認できる

詳しい手順は次の章でやります。

## ❌ よくあるミス

- Vite を「言語」だと思ってしまう
- Vite を入れれば自動で公開されると勘違いする
- 使う前に設定を自分で全部やらなきゃいけないと思って止まる
