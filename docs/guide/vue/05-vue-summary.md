# 第5章：Vue入門のまとめ

## ここまでで分かったこと

- Vueは画面づくりの道具
- 画面は部品（コンポーネント）の集まり
- データが画面に表示される
- ボタンなどで画面が動く
- Propsで部品にデータを渡せる

## ここまでできればOKのチェック

- `{{ }}` が「画面に出す場所」だと分かる
- `@click` で動きが作れる
- Propsでデータを渡せる

## 次に進むなら

- `v-for` と `v-if` の使い分け
- フォーム入力（`v-model`）
- API通信（`fetch`）

入門はここまででOKです。
