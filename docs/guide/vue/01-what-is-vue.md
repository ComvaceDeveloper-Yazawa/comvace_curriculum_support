# 第1章：Vueってなに？

## この章で身につくこと

- Vueが「画面づくりの道具」だと分かる
- 画面を「部品」で考える理由が分かる
- データと画面がつながる感覚が分かる

## 1-1 Vueは画面づくりの道具

Vueは、画面を作るためのライブラリです。
HTML/CSSだけだと「動かない画面」ですが、
Vueを使うと「動く画面」が作れます。

例えば、
- ボタンを押したら数字が増える
- 入力した文字がすぐ表示される

こういう動きが簡単に作れます。

## 1-2 画面は部品でできている

Vueでは画面を「部品」に分けます。
部品ごとに作ると、わかりやすくなります。

- ヘッダー
- メニュー
- カード

この1つ1つが「コンポーネント」と呼ばれます。

## 1-3 データと画面をつなぐ

Vueは「データ」と「画面」をつなげます。

```vue
<template>
  <p>{{ message }}</p>
</template>

<script setup lang="ts">
const message = "こんにちは";
</script>
```

`message` の中身が画面に表示されます。
<code v-pre>{{ }}</code> は「ここにデータを出す場所」です。

## 1-4 まずはここまででOK

- Vueは画面づくりの道具
- 画面は部品の集まり
- データが画面に出る

この感覚をつかめれば大丈夫です。
