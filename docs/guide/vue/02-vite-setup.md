# 第2章：ViteでVueを始める

## この章で身につくこと

- Vueプロジェクトの作り方が分かる
- 開発サーバーの起動方法が分かる

## 2-1 プロジェクトを作る

```bash
npm create vite@latest my-vue-app
```

- `my-vue-app` はフォルダ名です

このあと、選択画面で
「Vue」→「TypeScript」を選びます。

## 2-2 フォルダに移動

```bash
cd my-vue-app
```

作ったフォルダの中に入ります。

## 2-3 依存関係を入れる

```bash
npm install
```

必要な道具をまとめて準備します。

## 2-4 開発サーバーを起動

```bash
npm run dev
```

表示されたURL（例: `http://localhost:5173`）に
ブラウザでアクセスすると画面が見えます。

## 2-5 まずは「動いた」でOK

今は「Vueが動いた」を体験できれば十分です。
中身の意味は次の章からゆっくり学びます。
