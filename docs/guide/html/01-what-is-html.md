# 第 1 章：HTML とはなにか？

## この章で身につくこと

- HTML が「ページの中身・構造」を作る言語だと分かる
- タグで文章を挟むと画面に表示される仕組みが分かる
- HTML だけでも見た目に差が出る理由が分かる
- 次に CSS を学ぶ理由が分かる

## 1-1 HTML とはなにか？

HTML は、ページの「中身」と「構造」を作る言語です。
どこが見出しで、どこが本文で、どこがリストなのかを決めます。

ここでは「見た目をきれいにする話」はしません。
まずは「ページの骨組みを作るもの」だと覚えれば OK です。

## 1-2 Hyper Text Markup Language の説明

HTML は「Hyper Text Markup Language」の略です。
むずかしく聞こえますが、意味はシンプルです。

- Hyper Text: ほかのページへ飛べる文章（リンクのこと）
- Markup: 文章に「名札（タグ）」を付けること
- Language: そのためのルール

つまり HTML は「文章に名札を付けて、意味を伝えるルール」です。
今は「タグで囲むと意味が決まる」と分かれば十分です。

## 1-3 HTML だけで作った画面を見てみよう

まずは完成イメージを見ましょう。CSS なしでも、HTML だけで表示されます。

<DemoPreview html='<!doctype html><html lang="ja"><head><meta charset="UTF-8" /><title>HTMLだけのページ</title></head><body><h1>はじめてのHTML</h1><p>HTMLはページの中身を作るための言語です。</p><a href="#">リンクはこちら</a><ul><li>見出しがある</li><li>本文がある</li><li>リストがある</li></ul></body></html>' />

## 1-4 この画面の裏側ではこんなコードが書かれています

上の画面は、次の HTML だけで作られています。

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <title>HTMLだけのページ</title>
  </head>
  <body>
    <h1>はじめてのHTML</h1>
    <p>HTMLはページの中身を作るための言語です。</p>
    <a href="#">リンクはこちら</a>
    <ul>
      <li>見出しがある</li>
      <li>本文がある</li>
      <li>リストがある</li>
    </ul>
  </body>
</html>
```

## 1-5 HTML はページの基礎構造を作る

HTML は「ページの土台」を作る役目です。
設計図のように、どこに何があるかを決めます。

見出しは`h1`、本文は`p`、リストは`ul`と`li`のように書き分けます。
この「書き分け」が、ページの読みやすさにつながります。

## 1-6 HTML だけでは見た目はあまり変わらない

同じ文章でも、タグが違うと見た目が変わるのが分かります。
これは「タグごとに、初期の見た目が少し違う」からです。

まずはプレビューを見てください。

<DemoPreview html='<!doctype html><html lang="ja"><head><meta charset="UTF-8" /><title>タグの違い</title></head><body><h1>これはh1です</h1><p>これはpです</p><a href="#">これはaです</a><ul><li>これはliです</li></ul></body></html>' />

次に、同じ内容の HTML コードです。

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <title>タグの違い</title>
  </head>
  <body>
    <h1>これはh1です</h1>
    <p>これはpです</p>
    <a href="#">これはaです</a>
    <ul>
      <li>これはliです</li>
    </ul>
  </body>
</html>
```

HTML だけだと、見た目はまだ大きく変えられません。
見た目を整えるのは、次の章で学ぶ CSS の役割です。
今は「HTML で構造、CSS で見た目」と覚えておけば OK です。

## ❌ 初学者がよくやるミス

- 現象: 文字が出ない → 原因: タグで囲み忘れている → 直し方: `<p>文字</p>` のようにタグで挟む
- 現象: 画面がぐちゃぐちゃに見える → 原因: 見出しと本文を同じタグで書いている → 直し方: `h1`と`p`を使い分ける
- 現象: 見た目を変えたくて止まる → 原因: HTML だけで何とかしようとしている → 直し方: HTML は構造、見た目は CSS と割り切る

今は分からなくて OK です。
次の章で CSS を学ぶと、見た目の調整が一気にできるようになります。
