# 第3章：タグってなに？

## この章で身につくこと
- タグは「名札」のような役割だと分かる
- タグによって見た目と意味が変わると理解できる
- 正しいタグを使う理由が分かる

## タグとはなにか（名札の例え）
HTMLのタグは、文章に付ける「名札」です。
この名札によって、ブラウザが文章をどう扱うかが決まります。

たとえば「見出し」という名札なら見出しとして扱われ、
「リンク」という名札ならリンクとして扱われます。

## タグでブラウザの扱いが変わる
同じ文章でも、タグが違うと見た目や意味が変わります。
これはブラウザが「名札」を見て、表示のしかたを変えるからです。

## 同じ文章・違うタグのデモ
同じ文章を `h1` / `p` / `a` / `li` で比べると、違いがはっきり分かります。

```html
<!doctype html>
<html lang="ja">
<head>
  <meta charset="UTF-8" />
  <title>タグの違い</title>
</head>
<body>
  <h1>同じ文章です</h1>
  <p>同じ文章です</p>
  <a href="#">同じ文章です</a>
  <ul>
    <li>同じ文章です</li>
  </ul>
</body>
</html>
```

## 正しいタグを使う意味
正しいタグを使うと、読みやすくなり、意味が伝わります。
検索や読み上げでも正しく扱われるので、結果として親切なページになります。

## ❌ 初学者がよくやるミス
- 現象: 全部同じ見た目にしたくて同じタグばかり使う → 原因: タグの意味を意識していない → 直し方: 見出しは`h1`、本文は`p`など役割で分ける
- 現象: `li`だけを単体で使う → 原因: `ul`や`ol`を忘れている → 直し方: `li`は必ず`ul`や`ol`の中に入れる
- 現象: `a`をただの文字として使う → 原因: リンクの意味を知らない → 直し方: `a`は「移動したいとき」に使う
