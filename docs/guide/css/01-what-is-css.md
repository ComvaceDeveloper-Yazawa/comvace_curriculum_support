# CSSとはなにか？
〜HTMLで作った画面の見た目を整える技術〜

## 1. この章で身につくこと
- CSSが何をするものか分かる
- HTMLとの違いが分かる
- CSSを学ぶ意味が分かる

## 2. HTMLだけで作った画面はどう見える？
HTMLだけでも画面は表示されます。
文字は読めますが、見た目はとてもシンプルです。
「読めるけど、きれいではない」状態だと考えてください。

## 3. CSSは何をするもの？
CSSは、画面の見た目を整えるためのものです。
具体的には、次のようなことができます。

- 色をつける
- 文字の大きさを変える
- 余白をつける
- 画面を整える

ここでは小さなデモで確認します。

```html
<h1 class="title">CSSのはじめの一歩</h1>
<p class="text">見た目が変わると読みやすくなります。</p>
```

```css
.title { color: #e60033; }
.text { background-color: #f5f5f5; padding: 8px; }
```

## 4. HTMLとCSSの役割の違い（軽く）
HTMLは「内容と構造」を作るものです。
CSSは「見た目」を作るものです。

ここは深く覚えなくてOKです。
次の記事で、もう少し詳しく確認します。

## 5. CSSはあとからでも追加できる
HTMLをいちから書き直す必要はありません。
あとからCSSを足して、見た目だけを変えられます。

これはとても大きなメリットです。
「見た目の調整だけを後でできる」と覚えておけばOKです。

## 6. まとめ
CSSは「見た目」を担当します。
HTMLとセットで使うことで、読みやすく整った画面になります。

次は「CSSがどこまで担当するか」を学びます。
