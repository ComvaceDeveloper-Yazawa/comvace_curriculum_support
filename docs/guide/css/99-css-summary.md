# 第 6 章：CSS のまとめ

## 1. この章で身につくこと

- CSS でできることを整理できる
- HTML との役割分担を再確認できる
- 「ここまでで十分」と安心できる
- 次の制作フェーズに進める

## 2. CSS でやってきたことのおさらい

CSS で扱ってきたことは、大きくこの 3 つです。

- 色
- 文字
- 余白

## 3. HTML と CSS の役割まとめ

HTML は「部品」を作るものです。
CSS は「見た目・並び」を作るものです。

役割を分けるだけで、迷いが減ります。

## 4. よくある不安への回答

**覚えきれない**
最初は覚えきれなくて OK です。
よく使うものからで十分です。

**すぐ忘れる**
忘れても大丈夫です。
必要になったら調べれば OK です。

## 5. ❌ 初学者がよくやるミス

- 現象: 完璧を目指して手が止まる → 原因: 100 点を最初から狙う → 直し方: まずは 60 点で動かす
- 現象: コピペを悪だと思う → 原因: 自分で全部書かなきゃと思う → 直し方: 動く形を真似して覚える
- 現象: できない＝向いてないと思う → 原因: 成長途中だと忘れる → 直し方: 少しずつできることが増えていると確認する

## 6. 次にやること
次は HTML + CSS で 1 ページ作ってみましょう。

- 自己紹介ページ
- カード型レイアウト
- 簡単な LP（1 枚ものの紹介ページ）

```html
<div class="card">
  <h2>自己紹介</h2>
  <p>HTML で中身、CSS で見た目を整える。</p>
</div>
```

```css
.card { border: 1px solid #333; padding: 12px; }
```
